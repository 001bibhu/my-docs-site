# Setup and Integration

Follow these steps to start using ProductHelp AI.

## Step 1: Sign up

Create an account on the platform using your work email.

## Step 2: Connect your product

Integrate your product with ProductHelp AI using one of the following:

- JavaScript snippet (for web apps)
- API integration
- SDK (if available)

## Step 3: Start collecting data

Once integrated, ProductHelp AI will begin collecting:

- User search queries
- Product usage data
- Support interactions

## Step 4: Access dashboard

Log in and open the dashboard to start viewing insights.

## Step 5: Take actions

Use the insights to:

- Improve product experience
- Fix user issues
- Reduce support tickets

Setup is typically completed within a few minutes.
